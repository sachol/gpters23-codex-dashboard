## 결과물 요청 프롬프트

```text
아래 매물 자료를 공인중개사 실무 기준으로 정리해줘.
목표는 고객 설명용 브리핑 초안과 중개사 내부 검수 체크리스트를 만드는 것이다.

반드시 다음 기준을 지켜줘.
1. 주소, 면적, 가격, 용도, 층수, 사용승인일 등 수치는 입력자료 기준으로만 작성한다.
2. 자료 간 충돌이나 누락은 단정하지 말고 `추가 확인 필요`로 분리한다.
3. 건축물대장 API 결과는 원문 또는 다른 공적 자료와 교차검증이 필요하다고 표시한다.
4. 고객에게 전달할 문구와 내부 검수 메모를 분리한다.
5. 투자 수익, 권리 안전성, 적법성을 보장하는 표현을 쓰지 않는다.
6. 마지막에 이번 작업의 소요 시간, 자동화된 단계, 사람이 검수해야 할 단계를 표로 정리한다.
```
