이 문서는 `samples/property-briefing-sample-input.md`의 가상 자료를 기준으로 작성한 실습용 초안입니다. 실제 주소, 실제 권리관계, 실제 시세가 아니며 고객 제공용 자료로 사용할 수 없습니다.

## 2. 고객 설명용 브리핑 초안

이 매물은 입력자료 기준으로 서울시 샘플구 샘플동에 있는 매매가 30억 원의 근린상가 건물입니다. 대지면적은 220.00㎡, 연면적은 520.00㎡로 기재되어 있으며, 지하 1층부터 지상 4층까지의 건물로 정리되어 있습니다.

임대 현황 메모에 따르면 1층과 2층은 임대 중이고, 3층은 공실로 기재되어 있습니다. 4층은 실제 용도와 사용 현황을 추가 확인해야 합니다. 현장 메모상 역 출구에서 도보 7분이라는 설명과 1층 전면 노출이 좋다는 장점이 있으나, 실제 거리와 노출도는 현장 또는 지도 자료로 재확인하는 것이 좋습니다.

투자 검토 관점에서는 현재 임대 중인 층과 공실 층을 분리해서 보는 것이 필요합니다. 특히 3층 공실의 임대 가능성, 4층 실제 용도, 외벽 및 공용부 수리 이력, 등기부상 근저당권 내용은 고객 안내 전에 원문 자료로 확인해야 합니다.

## 3. 중개사 내부 검수 체크리스트

| 검수 항목 | 확인 기준 | 상태 | 메모 |
|---|---|---|---|
| 주소 | 건축물대장, 등기부, 토지이용계획 주소 일치 여부 | 추가 확인 필요 | 샘플 주소이므로 실제 원문 대조 없음 |
| 대지면적 | 건축물대장 표제부의 대지면적 기준 | 추가 확인 필요 | 220.00㎡ 입력값 사용 |
| 건축면적·연면적 | 건축물대장 원문 또는 API 결과 대조 | 추가 확인 필요 | 건축면적 130.00㎡, 연면적 520.00㎡ 입력값 사용 |
| 주용도 | 건축물대장 주용도 확인 | 추가 확인 필요 | 제2종근린생활시설 입력값 사용 |
| 용도지역 | 토지이용계획확인원 기준 확인 | 추가 확인 필요 | 제3종일반주거지역 메모 기준 |
| 위반건축물 여부 | 건축물대장 원문 확인 | 자료 없음 | 자료 없음은 위반 없음과 다름 |
| 권리관계 | 등기부등본 갑구·을구 확인 | 추가 확인 필요 | 근저당권 채권최고액 12억 원 가정 |
| 임대 현황 | 임대차계약서, 전입·사업자 현황 등 확인 | 추가 확인 필요 | 1층·2층 임대 중, 3층 공실, 4층 용도 확인 필요 |
| 수리 이력 | 매도인 진술, 영수증, 현장 확인 | 자료 없음 | 외벽·공용부 수리 이력 없음 |
| 광고 표현 | 과장·수익 보장·권리 안전성 단정 표현 제거 | 확인 필요 | 고객 제공 전 문구 재검토 |

## 4. 추가 확인 필요 항목

- 건축물대장 원문 기준 대지면적, 건축면적, 연면적, 주용도, 위반건축물 표시 확인
- 토지이용계획확인원 기준 용도지역과 기타 제한사항 확인
- 등기부등본 원문 기준 근저당권 채권최고액, 말소 예정 여부, 기타 권리 제한 확인
- 3층 공실 상태와 임대 가능 조건 확인
- 4층 실제 용도와 건축물대장상 용도 일치 여부 확인
- 외벽과 공용부 수리 이력 확인
- 역 출구 도보 7분, 1층 전면 노출 등 현장 메모의 객관 자료 확인

| 질문 | 답변 초안 |

| 공실은 몇 개 층인가요? | 입력자료 기준 3층은 공실로 기재되어 있고, 4층은 실제 사용 현황 추가 확인이 필요합니다. |
| 권리관계에 문제는 없나요? | 등기부 메모상 근저당권이 있는 것으로 되어 있어 원문 등기부와 말소 조건을 확인한 뒤 안내드려야 합니다. |
| 바로 임대 운영이 가능한가요? | 현재 임대 중인 층과 공실 층을 분리해서 확인해야 하며, 4층 용도와 공실 상태를 현장에서 다시 확인하는 절차가 필요합니다. |
