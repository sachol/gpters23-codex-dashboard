# 반복 업무 자동화 추천 메뉴

대표님의 `현직 중개업`, `법원경매 사업장 운영`, `공인중개사 AI 교육`을 한 프로젝트 안에서 연결하기 위한 Week 1 후보 목록입니다.

## 선택 기준

| 기준 | 의미 |
|---|---|
| 반복성 | 실제 업무에서 자주 발생하는가 |
| 시간절감 | 자료 확인·정리·초안 작성 시간을 줄일 수 있는가 |
| 입력자료 명확성 | 파일, 표, 메모, 링크 등으로 입력을 정리할 수 있는가 |
| 검수 가능성 | 대표님이 원문과 대조해 빠르게 오류를 확인할 수 있는가 |
| 사례 적합성 | GPTERS 발표와 공인중개사용 AI 강의로 전환하기 좋은가 |

이 조합은 대표님의 세 역할을 모두 보여주면서도, 매주 결과물 1개 이상을 발표해야 하는 공식 커리큘럼과 맞습니다.

## 공개·검수 기준

- 사건번호, 당사자명, 상세 주소, 연락처 등 식별정보는 공개 전 비식별 처리합니다.
- 매각물건명세서, 현황조사서, 감정평가서, 등기사항증명서 사이의 충돌은 숨기지 않고 별도 표시합니다.
- AI 결과는 권리 안전성, 적법성, 낙찰 가능성 또는 투자 수익을 보장하지 않습니다.
- 계산표에는 기준일, 출처, 가정값을 표시하고 실제 세금과 비용은 별도 확인합니다.

## 대표님이 추가할 값

각 후보의 `회당 소요 시간`, `월 발생 횟수`, `가장 번거로운 단계`를 알려주시면 시간절감 기대치와 2~4주 우선순위를 실측 기준으로 다시 계산합니다.
