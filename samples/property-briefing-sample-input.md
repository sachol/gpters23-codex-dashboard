# 비식별 샘플 매물 입력자료

이 파일은 Codex 앱 실습용 가상 자료입니다. 실제 주소, 실제 권리관계, 실제 시세가 아니며 고객 제공용 자료로 사용할 수 없습니다.

| 항목 | 내용 |
|---|---|
| 주소 | 서울시 샘플구 샘플동 100-1 |
| 거래 유형 | 매매 |
| 매도희망가 | 30억 원 |
| 매물 유형 | 근린상가 건물 |
| 대상 독자 | 법인투자자 |
| 출력 형태 | 고객 설명 초안 + 내부 검수 체크리스트 |

|---|---|---|

## 현장 메모

- 역 출구에서 도보 7분으로 안내받았습니다.
- 1층 전면 노출이 좋다는 설명을 들었습니다.
- 3층 공실 상태와 4층 실제 용도는 현장 재확인이 필요합니다.
- 외벽과 공용부 수리 이력은 자료가 없습니다.
- 매도인은 빠른 잔금을 선호한다고 들었습니다.
