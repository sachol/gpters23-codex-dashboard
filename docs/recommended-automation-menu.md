# 반복 업무 자동화 추천 메뉴

## 선택 기준

| 기준 | 의미 |
|---|---|
| 반복성 | 자주 발생해 자동화 효과가 큰가 |
| 입력자료 명확성 | Codex에게 줄 자료가 파일, 표, 메모, 링크로 정리 가능한가 |
| 검수 가능성 | 공인중개사가 결과를 빠르게 확인하고 수정할 수 있는가 |

| 법적 리스크 관리 | 단정 표현을 피하고 확인 필요 항목으로 분리 가능한가 |

## 1차 추천 조합

신화님 상황에서는 아래 3개 조합이 가장 좋습니다.

| 우선순위 | 추천 주제 | 이유 |
|---:|---|---|
| 1 | 공부서류 기반 매물 브리핑 초안 | 공인중개사 전문성이 가장 잘 드러나고, 에이전트 시스템으로 만들기 좋음 |
| 2 | 고객 상담 요약 및 후속 문자 | 반복성이 높고 즉시 실무 효율을 느끼기 좋음 |
| 3 | 공인중개사용 AI 프롬프트 모음 | 강의 활동과 직접 연결되고 수강생에게 제공 가능한 결과물이 됨 |

## 선택 방법

아래 문장으로 답하면 됩니다.

```text
내가 1차로 자동화하고 싶은 후보는 [번호/업무명]이야.
대략 회당 걸리는 시간은 [분/시간] 정도야.
이유는 [이유]야.
```

여러 개를 같이 하고 싶다면 이렇게 답하면 됩니다.

```text
1순위:
2순위:
3순위:
각각 대략 걸리는 시간:
```

## 판단 메모

- GPTERS Week 1 과제에는 주제 1개만 확정하면 충분합니다.
- 다만 최종 에이전트 시스템은 `중개업무 자동화 1개 + 강의 준비 자동화 1개` 조합으로 확장하면 신화님의 본업과 강사 활동을 모두 보여줄 수 있습니다.
- 시간 단축 수치는 반드시 실측 후 사례글에 반영합니다.
