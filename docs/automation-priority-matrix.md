# 자동화 후보 우선순위 평가표

이 표는 신화님의 업무 맥락을 바탕으로 만든 `가정 기반 초안`입니다. 실제 Week 1 과제 제출 전에는 최근 반복 업무 5개와 실제 소요 시간을 반영해 수정합니다.

## 평가 기준

각 항목은 1~5점으로 평가합니다.

| 기준 | 5점 의미 |
|---|---|
| 반복성 | 매주 반복되고 누락 시 업무 부담이 큼 |
| 시간 절감 기대 | Codex 적용 후 30분 이상 줄일 가능성이 큼 |
| 자료 준비 쉬움 | 파일, 링크, 메모 형태로 입력자료를 쉽게 모을 수 있음 |
| 검수 쉬움 | 사람이 결과를 빠르게 대조하고 수정할 수 있음 |
| 강의 사례 적합성 | 다른 공인중개사에게 설명하기 좋고 재사용 가능함 |
| 리스크 관리 가능성 | 법적·수치적 단정을 피하고 확인 항목으로 분리 가능함 |

## 1차 추천 유지 여부

총점만 보면 `공부서류 기반 매물 브리핑 초안`과 `고객 상담 요약 및 후속 문자 초안`이 같은 27점입니다. 그래도 1차 주제는 `공부서류 기반 매물 브리핑 초안`으로 유지하는 것이 좋습니다.

이유:

- 공인중개사 전문성이 더 잘 드러납니다.
- 건축물대장, 등기부, 토지이용계획, 실거래가처럼 명확한 입력자료가 있습니다.
- 고객 설명용 초안과 내부 검수 체크리스트를 분리하는 구조가 강의 사례로 좋습니다.
- GPTERS 최종 사례글에서 `AI에게 맡길 일`과 `공인중개사가 검수할 일`의 경계가 분명하게 보입니다.

## 후보별 Codex 적용 방식

| 후보 업무 | Codex에게 맡길 단계 | 사람이 검수할 단계 |
|---|---|---|
| 공부서류 기반 매물 브리핑 초안 | 자료 요약, 핵심 수치 추출, 고객 설명 초안, 내부 체크리스트 생성 | 원문 공부서류 대조, 권리관계 확인, 광고 표현 검수 |
| 고객 상담 요약 및 후속 문자 초안 | 상담 내용 요약, 니즈 분류, 후속 문자 초안 작성 | 개인정보 삭제, 고객 의도 확인, 발송 전 문구 수정 |
| 투자 검토 리포트 초안 | 수익률 계산 구조, 장단점 정리, 확인 필요 항목 생성 | 가정값 검토, 실제 시세 확인, 투자 권유 표현 제거 |
| 블로그·강의 콘텐츠 재가공 | 원고 구조화, 사례글 초안, 강의 목차 작성 | 사실관계 확인, 톤 조정, 공개 가능 정보 검토 |
| 매물 홍보 문구 생성 | 채널별 문안, 고객군별 카피, 금지 표현 체크 | 광고법·중개대상물 표시광고 기준 검수 |

## 신화님 확인 필요 항목

| 항목 | 확인할 내용 |
|---|---|
| 실제 반복 업무 5개 | 위 후보가 실제 최근 업무와 맞는지 |
| 업무별 회당 소요 시간 | 예상이 아니라 실제 체감 시간 또는 실측 시간 |
| 가장 부담되는 단계 | 자료 확인, 초안 작성, 검수, 고객 응대 중 어느 단계인지 |
| 공개 사례 가능성 | GPTERS 사례글에 공개 가능한 수준으로 비식별 처리 가능한지 |
| 1차 주제 유지 여부 | 매물 브리핑 자동화를 그대로 갈지, 상담 요약으로 바꿀지 |

## Week 1 제출용 한 줄 결론

후보를 비교한 결과, 저는 `공부서류 기반 매물 브리핑 초안 작성`을 1차 자동화 주제로 선택하려고 합니다. 이 업무는 반복성이 높고, 입력자료와 결과물이 분명하며, 공인중개사가 최종 검수해야 할 항목을 명확히 분리할 수 있어 Codex 앱 실험과 강의 사례 모두에 적합하다고 판단했습니다.
