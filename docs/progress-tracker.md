| 리스크 | 관리 방법 |
|---|---|
| 수치 오류 | 입력자료 원문과 대조, 단위(㎡/평) 구분 |

| 개인정보·매물 식별정보 노출 | 사례글 공개 전 주소, 소유자, 연락처 비식별 처리 |
| AI 결과 과신 | 고객 전달 문구와 내부 검수 메모 분리 |
| 사례글 근거 부족 | 실제 명령, 소요 시간, 산출물 파일을 증거 장부에 기록 |
